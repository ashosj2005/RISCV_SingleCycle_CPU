# RISC-V Single Cycle Processor

## Overview
This project implements a basic RISC-V Single Cycle Processor in Verilog HDL.

## Features
- Program Counter (PC)
- Instruction Memory
- Register File
- ALU
- Control Unit
- Single-Cycle Execution

## Tools
- Verilog HDL
- Icarus Verilog
- GTKWave
- Ubuntu Linux

## Project Status
🚧 Under Development

## Repository Structure

src/ -> RTL Modules
tb/ -> Testbenches
docs/ -> Architecture Diagrams
results/ -> Simulation Results
